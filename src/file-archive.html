1.RDP клиенты (Программы для подключения через удаленный рабочий стол)

Microsoft RDP Client for MAC OS Требуется авторизация

Официальный и рекомендованный RDP клиент для операционной системы MAC OS от компании Microsoft.

Microsoft Remote Desktop для Android Требуется авторизация

Официальный клиент удаленного рабочего стола (RDP) от компании Microsoft для смартфонов / планшетов на базе ОС Android.

Microsoft Remote Desktop для iOS Требуется авторизация

Официальный клиент удаленного рабочего стола (RDP) от компании Microsoft для смартфонов / планшетов на базе iOS.

RDP клиент v6.1 для Windows XP SP2 Требуется авторизация

Обновление клиента удаленного рабочего стола (RDP) для Windows XP SP2. Обязательно установите, если у вас RDP версии 5.1 или ниже. Если у вас Windows XP SP3 - вам нужно установить ДРУГОЕ обновление!

RDP клиент v8.0 для Windows 7 Требуется авторизация

Обновление для протокола удаленного рабочего стола (RDP) версии 8.0 для ОС Windows 7 и Windows 7 SP1, позволяет использовать новые возможности служб удаленных рабочих столов.

Обновленный RDP клиент (WM v6.5) для Windows Mobile 5 и 6.1

Установите обновленный RDP клиент из архива. Если при запуске появляется ошибка "wpctsc не является допустимым приложением для pocket PC" - попробуйте установить MS_RDP_v6.5.cab

Расширение RDP Plus для RDP клиента под Windows Требуется авторизация

Надстройка для стандартного RDP клиента Windows, предоставляющая дополнительные возможности (сохранение нескольких логинов для одного IP и т.д.)

RDP клиент v2.1.1 для Mac OS X 10.5.8 - 10.6.x Требуется авторизация

Клиент (v. 2.1.1) подключения к удаленному рабочему столу (RDP) для Mac OS X 10.5.8 или более поздней версии, позволяет подключить компьютер Macintosh к компьютеру под управлением Windows и работать с приложениями и файлами, расположенными на нем. НЕ поддерживает систему Mac OS X 10.7 (Lion) и более поздних версий.

RDP клиент v7.0 для Windows XP SP3 Требуется авторизация

Обновление клиента удаленного рабочего стола (RDP) для Windows XP SP3. Обязательно установите, если у вас RDP версии 6.1 или ниже. Обновление подходит только для Windows XP SP3 (на систему с SP2 установить его нельзя).

2.Полезные утилиты

Программы для диагностики сети и т.д.

credssp-fix.reg Требуется авторизация

REG файл для исправления ошибки доступа "Произошла ошибка при проверке подлинности. Указанная функция не поддерживается"

WinMTR

Windows утилита для диагностики сети и выявления проблемных узлов (на которых пропадают пакеты) между вашим и удаленным сервером. Внимание: для анализа необходим отчет не менее чем из 500 пакетов!

Обновленный RDP клиент (WM v6.5) для Windows Mobile 5 и 6.1

Установите обновленный RDP клиент из архива. Если при запуске появляется ошибка "wpctsc не является допустимым приложением для pocket PC" - попробуйте установить MS_RDP_v6.5.cab
